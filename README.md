# Caderno Temático com NotebookLM: Estrutura de Dados II

## 1. Contexto e Objetivos

Este projeto foi desenvolvido como parte do desafio prático da DIO, com o objetivo de utilizar Inteligência Artificial como ferramenta de aprendizagem ativa, organização de conteúdo e apoio aos estudos.

O tema escolhido foi **Estrutura de Dados II**, uma disciplina essencial para a formação em desenvolvimento de software, pois apresenta estruturas fundamentais para armazenar, organizar, buscar e manipular dados de forma eficiente.

Durante o projeto, utilizei o NotebookLM para organizar materiais de estudo, revisar conceitos, gerar resumos, criar perguntas de revisão e estruturar um miniguia sobre os principais conteúdos da disciplina.

## Objetivos de estudo

- Compreender os principais conceitos de Estrutura de Dados II.
- Revisar listas encadeadas, pilhas, filas, árvores, árvores AVL, tabelas hash e métodos de ordenação.
- Entender a lógica por trás das estruturas e suas aplicações.
- Relacionar conceitos teóricos com exemplos de código.
- Criar um material de revisão organizado para estudos futuros.
- Utilizar Engenharia de Prompts para extrair melhores respostas da IA.

## 2. Curadoria de Fontes

As fontes utilizadas para alimentar o NotebookLM foram selecionadas com base nos conteúdos estudados na disciplina de Estrutura de Dados II.

### Fontes utilizadas

1. Material da disciplina sobre **Lista Encadeada**  
2. Material da disciplina sobre **Pilha e Fila**  
3. Material da disciplina sobre **Árvores Binárias**  
4. Material da disciplina sobre **Árvore AVL**  
5. Material da disciplina sobre **Tabela Hash e Métodos de Pesquisa/Ordenação**

### Critérios de escolha das fontes

As fontes foram escolhidas porque abordam os principais conteúdos da disciplina, apresentam conceitos fundamentais e servem como base para entender tanto a parte teórica quanto a implementação em código.

Os materiais também foram úteis para construir resumos, glossário, exemplos e perguntas de revisão.
